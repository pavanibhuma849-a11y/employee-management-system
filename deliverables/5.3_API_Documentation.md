## Employee API
Base URL: `/employees`

### 1. Create Employee
- **URL**: `/create`
- **Method**: `POST`
- **Description**: Creates a new employee record and associates them with a department and projects.
- **Request Example**:
```json
{
  "name": "John Doe",
  "role": "Software Engineer",
  "salary": 75000.0,
  "joiningDate": "2023-01-15",
  "departmentId": 1,
  "projectIds": [1, 2]
}
```
- **Response Example (200 OK)**:
```json
{
  "id": 1,
  "name": "John Doe",
  "role": "Software Engineer",
  "salary": 75000.0,
  "joiningDate": "2023-01-15",
  "departmentName": "Engineering",
  "projectNames": ["Cloud Migration", "Internal Tool"]
}
```

### 2. Get Employee by ID
- **URL**: `/{id}`
- **Method**: `GET`
- **Description**: Retrieves details of a specific employee by their unique ID.
- **Response Example (200 OK)**:
```json
{
  "id": 1,
  "name": "John Doe",
  "role": "Software Engineer",
  "salary": 75000.0,
  "joiningDate": "2023-01-15",
  "departmentName": "Engineering",
  "projectNames": ["Cloud Migration"]
}
```

### 3. List Employees
- **URL**: `/`
- **Method**: `GET`
- **Description**: Returns a paginated list of employees. Can be filtered by department.
- **Query Parameters**:
  - `department` (optional): Filter by department name.
  - `page` (default: 0): Page number.
  - `size` (default: 10): Items per page.
- **Response Example (200 OK)**:
```json
{
  "content": [
    { "id": 1, "name": "John Doe", ... }
  ],
  "pageable": { ... },
  "totalPages": 1,
  "totalElements": 1
}
```

### 4. Get Sorted Employees
- **URL**: `/sorted`
- **Method**: `GET`
- **Description**: Returns all employees sorted by name and joining date.
- **Response Example (200 OK)**:
```json
[
  { "id": 1, "name": "Alice Johnson", ... },
  { "id": 2, "name": "Bob Smith", ... }
]
```

### 5. Update Employee
- **URL**: `/{id}`
- **Method**: `PUT`
- **Description**: Updates the details of an existing employee.
- **Request Example**:
```json
{
  "salary": 80000.0,
  "role": "Senior Software Engineer"
}
```

### 6. Delete Employee
- **URL**: `/{id}`
- **Method**: `DELETE`
- **Description**: Removes an employee record from the system.
- **Response**: `204 No Content`

## Department API
Base URL: `/departments`

### 1. Create Department
- **Method**: `POST`
- **URL**: `/`
- **Request**: `{ "name": "Finance" }`
- **Response**: `{ "id": 4, "name": "Finance" }`

### 2. Get All Departments
- **Method**: `GET`
- **URL**: `/`
- **Response**: `[{ "id": 1, "name": "HR" }, { "id": 2, "name": "Engineering" }]`

### 3. Update Department
- **Method**: `PUT`
- **URL**: `/{id}`
- **Request**: `{ "name": "Talent Acquisition" }`
